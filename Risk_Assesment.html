위험성평가 자동화 시스템 사용 매뉴얼

1. 시스템 개요

본 프로그램은 건설 현장의 [최초/상시] 위험성평가 업무를 자동화하여 엑셀 문서 작성 시간을 획기적으로 단축해 주는 시스템입니다. 공종/작업별 위험요인을 클릭만으로 쉽게 선택하고, 자동으로 표준화된 엑셀 서식에 맞게 분리 출력할 수 있습니다.

2. 평가 유형 및 기본정보 입력

  • 평가 유형: 화면 우측 상단에서 [최초] 평가 또는 [상시] 평가를 선택합니다. 선택한 유형에 맞는 엑셀 템플릿이 자동으로 적용됩니다.
  • 기본정보: 공사명, 시공사(기본값 적용됨), 현장소장, 작성기준일을 입력합니다.
  • Tip: 작성기준일은 숫자 8자리(예: 20260324)만 연속으로 입력하면 자동으로 YYYY-MM-DD 형식으로 변환됩니다.

3. 평가요인 검색 (필터 적용)

  • 공종명을 먼저 선택합니다.
  • 활성화된 작업명을 선택합니다. (필요 시 단위작업명까지 상세하게 필터링할 수 있습니다.)
  • 우측의 [필터 적용] 버튼을 클릭하면, 하단 리스트에 해당 조건의 평가요인들이 즉시 검색됩니다.
  • Tip: 검색 단계에서 이미 동일한 [공종명 + 기인물 + 재해유발요인]은 스마트하게 중복이 제거되어 깔끔하게 1줄로 표시됩니다. (엑셀 출력 시 숨겨진 세부 감소대책 등은 온전하게 반영됩니다.)

4. 기인물 다중 필터 활용 (선택사항)

필터 적용 후, 검색된 항목이 너무 많을 경우 [기인물 다중 필터] 토글 버튼이 나타납니다.

  • 원하는 기인물 버튼을 클릭하여 해당하는 항목만 좁혀서 확인할 수 있습니다.
  • 여러 개를 동시 선택(다중 클릭)하여 원하는 기인물 묶음만 띄워놓고 체크할 수 있습니다.
  • 선택을 해제하거나 [전체 기인물]을 누르면 리스트가 원래대로 모두 표시됩니다.

5. 평가요인 최종 추가

  • [검색된 평가요인] 리스트에서 현장에 해당하는 요인의 체크박스를 선택합니다.
  • [선택요인 아래로 추가] 버튼을 누르면, 체크한 항목들이 하단의 [선택된 최종 평가요인] 목록으로 복사됩니다.
  • 기인물 필터(탭)가 가려져 있어 화면에 당장 보이지 않더라도, 체크해 둔 항목은 메모리에 보존되어 한 번에 모두 합산되어 추가됩니다.
  • 이미 아래에 추가된 요인을 실수로 다시 넣더라도 시스템이 자동으로 중복을 걸러줍니다.

6. 엑셀 출력 및 데이터 관리

  • 엑셀 출력: 모든 요인을 하단 리스트에 담은 후, 우측 상단의 [엑셀 출력] 버튼을 누릅니다.
    ▶ 프로그램이 템플릿 서식(셀 병합, 테두리, 인쇄 영역 등)을 완벽하게 유지하며, 각 공종별로 시트를 분리하여 단일 엑셀 문서로 자동 생성합니다.
  • 파일 저장/열기: 진행 중이거나 자주 쓰는 현장 세팅을 [파일 저장] (JSON 포맷) 해두었다가 언제든 [파일 열기]로 복구하여 이어할 수 있습니다.
