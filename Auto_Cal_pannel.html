 판넬계산기 사용 매뉴얼

                
            

            

                

                    
1. 진행 순서 요약

                    
    
                        
  1. 기본정보 설정: 길이, 높이 변수 및 지붕형태, 명칭 태그, 로스율을 입력합니다.
    2. 
                        
  2. 제외 창호 등록: 자주 쓰이는 창호의 규격을 미리 데이터베이스에 등록합니다.
    3. 
                        
  3. 케이스 자동 산출: "자동산출하기" 버튼을 눌러 지붕형태에 맞는 리스트를 생성합니다.
    4. 
                        
  4. 제외 및 수동입력: [제외 항목 입력] 버튼으로 창호를 빼거나, [수동입력] 버튼으로 개별 케이스를 추가합니다.
    5. 
                        
  5. 데이터 내보내기: 작업이 끝나면 Excel이나 JSON으로 저장하여 보관합니다.
    6. 
                    

                

                

                    
2. 주요 기능 및 팁 (Tip!)

                    
    
                        
  • 💡 
    산식 확인 (더블클릭): 산출 리스트 표의 결과 셀을 더블클릭하시면 팝업으로 상세 산식을 확인할 수 있습니다.
    • 
                        
  • ➖ 
    제외 항목 입력: 각 행 아래에 있는 제외 항목 입력 버튼을 누르면 해당 구역에서 뺄 창호나 배관 면적을 무제한으로 추가할 수 있습니다.
    • 
                        
  • 💾 
    데이터 백업: JSON 저장 버튼을 눌러 파일로 백업하고, 나중에 JSON 열기로 이어서 작업할 수 있습니다.
    • 
                    

                

                

                    
3. 지붕 형태별 계산식 (단순 산출값)

                    

                        
* 모든 입력값은 밀리미터(mm) 기준이며, 수식 내부에서 1000으로 나누어 계산 후 자동 환산됩니다.

                        
                        
▶ 1면경사지붕

                            
    
                                
  • 정/배면: 장변 * 처마높이
    • 
                                
  • 좌/우측면: 단변 * (최고높이 + 처마높이) * 0.5
    • 
                                
  • 지붕: (SQRT((최고높이-처마높이)² + 단변²) + 0.3) * 장변
    • 
                                
  • 파라펫: (최고높이 - 처마높이) * (장변 + 단변)
    • 
                            

                        

                        
▶ 2면경사지붕

                            
    
                                
  • 정/배면: 장변 * 처마높이
    • 
                                
  • 좌/우측면: 단변 * 처마높이 + (최고높이 - 처마높이) * 단변 * 0.5
    • 
                                
  • 지붕: (SQRT((최고높이-처마높이)² + (단변/2)²) + 0.6) * 2 * 장변
    • 
                            

                        

                        
▶ 평지붕

                            
    
                                
  • 측벽(4면): 해당 변길이 * 처마높이
    • 
                                
  • 지붕: 단변 * 장변
    • 
                            

                        

                    

                

                

                    
4. 순면적 및 로스율 계산식

                    

                        
순면적단순산출값 - 제외면적 합계

                        
로스적용순면적 × (1 + 로스율)
