마당포장 계산기 사용 매뉴얼

1. 예상 면적 입력

상단의 예상 평수 또는 예상 면적(M²) 입력칸에 원하는 숫자를 입력하세요. 평수와 M²는 1평 = 3.3M² 기준으로 양방향 자동 변환됩니다.

2. 실행 외주가 확인 (상세 내역)

입력된 면적을 바탕으로 아스콘/콘크리트 포장에 필요한 자재물량, 장비, 노무비가 즉시 계산되어 중간 표에 표시됩니다. (수량 × 단가)

3. 단가 및 기본 설정 변경

우측 상단의 [단가/설정 변경] 버튼을 누르시면 자재비, 장비비, 투입 인원, 할증률 등을 현장 상황에 맞게 직접 수정할 수 있습니다.

4. 제출용 내역서 (평/M² 역산)

맨 아래의 자동 산출 표는 총 실행금액을 면적으로 다시 나누어(역산), 실제 관공서나 고객에게 제출하기 좋은 단위 면적당 단가표를 제공합니다.
