  1. 기본정보 설정: 전체 프로젝트에 적용될 높이와 로스율을 입력합니다.
    2. 
                        
  2. 제외 창호 등록: 자주 쓰이는 창호의 규격을 미리 데이터베이스에 등록합니다.
    3. 
                        
  3. 산출 리스트 입력: 구역별로 가로길이 및 제외 창호를 선택해 리스트에 추가합니다.
    4. 
                        
  4. 데이터 내보내기: 작업이 끝나면 Excel이나 JSON으로 저장하여 보관합니다.
    5. 
                    

                

                

                    
2. 주요 기능 및 팁 (Tip!)

                    
    
                        
  • 💡 
    산식 확인 (더블클릭): 산출 리스트 표에서 단순산출, 제외합계, 최종면적, 로스적용 셀을 더블클릭하시면 팝업으로 상세 산식을 확인할 수 있습니다.
    • 
                        
  • ➕ 
    동적 입력란 추가: 입력 폼 내의 [+ 창호 추가] 또는 [+ 수동입력 추가] 버튼을 눌러 칸을 무한히 늘릴 수 있습니다.
    • 
                        
  • 💾 
    데이터 저장/불러오기: JSON 저장 버튼을 눌러 파일로 백업하고, 나중에 JSON 열기로 이어서 작업할 수 있습니다.

3. 지붕 형태별 계산식 (단순 산출값)

                    

                        
* 모든 입력값은 밀리미터(mm) 기준이며, 최종 결괏값은 제곱미터(m²)로 자동 환산(÷ 1,000,000) 됩니다.

                        
▶ 상관없음
가로길이(L) × 처마높이

                        
▶ 1면경사지붕(사다리꼴)
(처마높이 + 최고높이) × 가로길이(L) × 0.5

                        
▶ 2면경사지붕(5각형)
(처마높이 × 가로길이) + ((최고높이 - 처마높이) × 가로길이 × 0.5)

                    

                

                

                    
4. 최종 면적 및 로스율 계산식

                    

                        
최종 면적단순산출값 - (등록창호 제외면적 합계 + 수동 입력 제외면적 합계)

                        
로스적용최종 면적 × (1 + (전체 로스율 % / 100))
